pkuDiss（PKU Dissertations） 是一套面向北京大学研究生学位论文的 \LaTeX{} 模板。
模板遵循研究生院学位论文写作指南，并基于先进制造与机器人学院的格式审查完成适配。
简要使用说明如下：

（1） 模板主文件为 \texttt{main.tex}，构建封面、正文、参考文献、附录、个人简历、致谢等学位论文主体结构，相应文件置于 \texttt{data} 目录。

（2） 插图及附页置于 \texttt{figures} 文件夹内，如封面北京大学图标 \texttt{figures/logo.eps}、版权声明 \texttt{figures/copyright.pdf} 以及原创性声明页 \texttt{figures/Authority.pdf}。

（3） 参考文献基于 BibTeX 生成，相关文献代码置于 \texttt{data/reference.bib}。

（4） \LaTeX{} 编译可采用本地或云端方式，如 {\color{red}{ TeX Live + VS Code + LaTeX Workshop 本地编译（推荐）}} 或 Overleaf 云端编译，详见 \texttt{README.md} 相关说明。

（5） 建议采用 pdf 格式插图，可显著提高编译效率，并确保插图清晰。
